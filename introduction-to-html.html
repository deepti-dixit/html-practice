Introduction to HTML

What is HTML?

HTML (HyperText Markup Language) is the standard language used to create web pages. It defines the structure of a web page by using a series of elements or tags.

HTML is essential for web development because it allows developers to structure content, embed images, videos, and links, and create interactive forms.

Basic Structure of an HTML Document

Every HTML document has a standard structure that includes the following elements:

  • <!DOCTYPE html> - Declares the document type as HTML5.
  • <html> - The root element of the HTML document.
  • <head> - Contains metadata such as the document's title, charset, and links to external resources.
  • <body> - Contains the content of the web page, such as headings, paragraphs, images, and links.

Example of a Basic HTML Document

Below is an example of a simple HTML document:

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My First HTML Page</title>
  </head>
  <body>
    <h1>Hello, World!</h1>
    <p>This is my first HTML page.</p>
  </body>
</html>

Why Learn HTML?

HTML is the backbone of all websites. Without HTML, web browsers would not know how to display content. By learning HTML, you can create static web pages and also understand the foundation needed for dynamic web development.
